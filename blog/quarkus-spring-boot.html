Quarkus vs Spring Boot, heeft het nieuwe framework een kans?

Spring boot is op het moment een standaard voor het bouwen van java (web) applicaties. Een applicatie bouwen zonder spring boot te gebruiken is bijna ondenkbaar. Een framework dat zo dominant is heeft natuurlijk voor- en nadelen. Zo kennen bijna alle java ontwikkelaars spring boot en hoe zo'n applicatie in elkaar zit.

Wat is Quarkus?

Quarkus is een alternatief voor Spring boot die zich voornamelijk richt op de Cloud native applicaties en Kubernetes. Zo komt een Quarkus applicatie standaard met een docker image en een Kubernetes manifest na het bouwen. Het heeft ook laten zien dat het bouwen op GraalVM grote voordelen heeft voor deze doeleinden. Spring boot heeft als gevolg ook de GraalVM ondersteuning toegevoegd, via spring-native.
